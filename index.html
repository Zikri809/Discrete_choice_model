Input Rules

Each respondent follows the same flow before moving to the next person:

  1. Rate all transport methods on a 1 to 5 scale.
  2. Rank the four attributes using each rank from 1 to 4 exactly once.

Cost is treated as a penalty in the utility model. The other attributes are benefits.
